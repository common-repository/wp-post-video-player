WP Post Video Player allows visitor to Play Video File which is assign to post.

== Description ==

WP Post Video Player plugin allows visitor to Play video on post page.

WP Post Video Player has number of options to configure.

You can change configuration setting for video player.

You must configure ffmpeg path before upload video.If you use linux server just write ffmpeg for ffmpeg path

In admin side you can Upload Video File And after that you can assign this file to Post

You can also  use this plugin for Video websites. You can describe Video with the post and assign file

This Plugin use for Video testimonial.

You can change Place of VideoPlayer By changing in Wordpress Hook.

By using `[video-player]` short code  in Post or page for display video player

For detailed documentation visit http://www.mitulhirpara.com/wp-post-video-player/

== Installation ==

1. Upload to the "wp-post-video-player" directory to `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure plugin
